# HR 4.0 #1 – Pošlji CV za pametno uvajanje z AI in Microsoft 365

## Uvod

Oddelek za človeške vire je pogosto preplavljen z ročnimi in ponavljajočimi se nalogami: pregledovanje življenjepisov, zbiranje dokumentov za zaposlitve, priprava kontrolnih seznamov in materialov za uvajanje.
V konkurenčnem okolju, kjer čas zaposlitve pomeni razliko, lahko ti počasni procesi ogrozijo sposobnost privabljanja in zadrževanja najboljših talentov.

Z novimi tehnologijami pa je mogoče obrniti paradigmo. V tem prvem poglavju serije *HR 4.0* si bomo ogledali, kako lahko **AI development** in **Microsoft 365** združimo za preoblikovanje upravljanja s prijavami in uvajanjem, kar omogoča, da je postopek hiter, standardiziran in skalabilen.

## Rešitev: AI + Microsoft 365

Ključ je kombinirati **intelligenza artificiale** za razčlenjevanje CV-jev z ekosistemom **Microsoft 365**, še posebej **SharePoint** in **Teams**.

### Korak 1 – Parsiranje CV-jev z AI

Model AI (na primer GPT-4.1 ali ekvivalenti) analizira življenjepise v PDF/Word in iz njih izvleče strukturirane informacije:

* tehnične kompetence,
* delovne izkušnje,
* izobraževanje,
* mehke veščine.

Rezultat je **JSON strukturiran** enostavno integrabilen v baze podatkov ali sezname SharePoint.

### Korak 2 – Ustvarjanje kartic kandidatov v SharePoint

Vsak analiziran CV postane **digitalna kartica** v seznamu SharePoint, s predhodno izpolnjenimi polji:

* Ime, vloga, izkušnje,
* Ključne kompetence,
* AI ocena ujemanja glede na delovni profil.

### Korak 3 – Portale Onboarding

Ko je kandidat izbran, SharePoint postane tudi **portal za uvajanje**:

* standardni dokumenti na voljo v OneDrive,
* kontrolni seznam nalog (npr. IT dostopi, identifikacijske kartice, usposabljanje),
* digitalni obrazci za podjetniške politike,
* namenski kanal Teams za sprejem novega zaposlenega.

## Tehnični primer: AI za obdelavo življenjepisov

Per prikaz bolj konkretnega dela, tukaj je **primer v Pythonu**, ki uporablja API OpenAI za pretvorbo CV-ja v PDF v strukturiran JSON:

```python
import openai
from PyPDF2 import PdfReader

# Leggi testo da CV PDF
reader = PdfReader("cv_candidato.pdf")
text = "".join([page.extract_text() for page in reader.pages])

# Chiamata API per estrarre dati
response = openai.ChatCompletion.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": "Estrai competenze, esperienze e formazione da un CV. Rispondi in JSON."},
        {"role": "user", "content": text}
    ],
    temperature=0.2
)

print(response.choices[0].message["content"])
```



Očekovani izhod (poenostavljen primer):

```json
{
  "nome": "Mario Rossi",
  "ruolo": "Data Analyst",
  "competenze": ["SQL", "Power BI", "Python"],
  "esperienze": [
    {"azienda": "XYZ Spa", "anni": 3, "ruolo": "Business Analyst"}
  ],
  "formazione": "Laurea in Economia"
}
```
